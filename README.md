# PCP Claims Landing Page

FCA PS26/3 compliant motor finance claims landing page.

Built with Next.js 14 · TypeScript · Tailwind CSS · React Hook Form · Zod

## Features
- 4-step claim wizard with eligibility screening
- DVLA vehicle registration lookup
- PS26/3 compliant copy and fee disclosures
- FCA free-route notices throughout

## Setup
```bash
npm install
cp .env.local.example .env.local  # add DVLA API key
npm run dev
```
